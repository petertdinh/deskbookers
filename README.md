```
	> git clone https://github.com/petertdinh/deskbookers
	> cd deskbookers
	> npm install
	> npm start
	> browse to http://localhost:8080/
```

To run tests

```
	> npm run test
```
The boilerplate that I used is Stephen Grider's ReduxSimpleStarter. I removed Redux as the application didn't need it.
